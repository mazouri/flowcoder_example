# 植物大战僵尸前端 产品需求文档 (PRD)

## 1. 概述

| 项目 | 说明 |
|------|------|
| **版本** | v1.0 |
| **功能分类** | 休闲塔防游戏 |
| **实现方式** | 纯前端 Web 应用（HTML5 Canvas / WebGL + JavaScript/TypeScript） |
| **状态** | 草稿 |
| **目标平台** | 现代浏览器（Chrome、Firefox、Safari、Edge） |

---

## 2. 背景与目标

### 2.1 背景

- **经典 IP 复刻**：植物大战僵尸（Plants vs Zombies）是 PopCap 于 2009 年发行的经典塔防游戏，具有广泛的用户认知度和情怀价值
- **技术验证**：作为前端技术实践项目，可验证 Canvas/Web 游戏开发、状态管理、动画与碰撞检测等核心能力
- **用户需求**：用户希望在浏览器中无需安装即可体验经典塔防玩法

### 2.2 目标

| 类型 | 目标 |
|------|------|
| **业务目标** | 打造可玩性高、体验流畅的 Web 版植物大战僵尸，作为技术展示或教学演示 |
| **用户目标** | 在浏览器中完成种植植物、抵御僵尸、通关关卡的核心游戏体验 |
| **技术目标** | 纯前端实现，无后端依赖，支持响应式布局，适配不同屏幕尺寸 |

### 2.3 成功指标

| 指标 | 目标值 | 衡量方式 |
|------|--------|----------|
| 首屏加载 | < 3s | 性能监控 |
| 帧率 | ≥ 30 FPS | 游戏运行时 |
| 关卡完成率 | 用户可通关至少 1 关 | 功能验收 |
| 兼容性 | 主流浏览器 4 款 | 兼容性测试 |

---

## 3. 功能需求

### 3.1 核心功能

#### 3.1.1 游戏画布与布局

- **输入**：无（自动渲染）
- **处理**：
  - 5 行 × 9 列草坪网格
  - 左侧为植物选择区（种子包），右侧为僵尸出生区
  - 阳光资源显示（左上角）
- **输出**：可视化游戏界面

#### 3.1.2 阳光系统

- **输入**：时间流逝（自动掉落）、阳光掉落
- **处理**：阳光从天上掉落，用户点击收集；阳光用于购买植物
- **输出**：阳光数量实时更新

#### 3.1.3 植物系统

| 植物 | 阳光消耗 | 功能 | 冷却时间 |
|------|----------|------|----------|
| 向日葵 | 50 | 生产阳光 | 7.5s |
| 豌豆射手 | 100 | 发射豌豆攻击 | 7.5s |
| 坚果墙 | 50 | 高血量阻挡 | 30s |
| 樱桃炸弹 | 150 | 范围爆炸 | 50s |

- **输入**：用户点击种子包 → 点击草坪格子
- **处理**：检查阳光是否足够、格子是否为空、冷却是否结束
- **输出**：植物被种植到指定格子

#### 3.1.4 僵尸系统

| 僵尸类型 | 血量 | 速度 | 特性 |
|----------|------|------|------|
| 普通僵尸 | 普通 | 慢 | 基础行走 |
| 路障僵尸 | 高 | 慢 | 戴路障，血量更高 |

- **输入**：波次触发（每关固定波次）
- **处理**：僵尸从右侧生成，沿行向左移动
- **输出**：僵尸到达终点则游戏失败

#### 3.1.5 战斗逻辑

- **输入**：植物与僵尸的碰撞检测
- **处理**：
  - 豌豆攻击僵尸，扣减僵尸血量
  - 僵尸啃食植物，扣减植物血量
  - 僵尸死亡后移除
  - 植物死亡后格子清空
- **输出**：实时战斗反馈

#### 3.1.6 关卡与胜负

- **输入**：波次完成、僵尸数量、存活植物
- **处理**：
  - 所有波次僵尸全部消灭 → 胜利
  - 任一僵尸到达最左侧 → 失败
- **输出**：胜利/失败界面，可重新开始

### 3.2 功能边界

| 支持 | 不支持 |
|------|--------|
| 单关卡模式（默认 1 关） | 多关卡存档 |
| 4 种植物 + 2 种僵尸 | 完整原版植物/僵尸种类 |
| 鼠标点击操作 | 触屏手势、键盘快捷键 |
| 本地运行 | 用户登录、排行榜、云存档 |

### 3.3 边界条件

- **数据限制**：阳光上限 9999，单格仅 1 株植物
- **格式要求**：资源需支持 Web 格式（PNG/SVG 等）
- **异常处理**：网络异常时使用占位图；窗口失焦时暂停游戏

## 4. 技术依赖

| 类型 | 说明 |
|------|------|
| **核心技术** | HTML5、CSS3、JavaScript/TypeScript |
| **渲染方案** | Canvas 2D 或 Pixi.js/Phaser 等 2D 游戏引擎 |
| **构建工具** | Vite / Webpack |
| **第三方服务** | 无（纯前端，资源可本地或 CDN） |

---

## 5. 排除范围（本期不做）

- 音效与背景音乐
- 多语言支持
- 移动端触屏优化
- 用户成就系统
- 内购/付费
- 多人在线对战
